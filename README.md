# 🌙 MOONLIGHT GARDEN — Женский клуб

Веб-приложение для бронирования столиков в женском ночном клубе.

## 🛠 Технологии
- PHP (бэкенд)
- MySQL (база данных)
- HTML5, CSS3, JavaScript (фронтенд)
- XAMPP (локальный сервер)

## 🚀 Установка и запуск

1. Установите [XAMPP](https://www.apachefriends.org/)
2. Скопируйте папку `moonlight-garden` в `C:\xampp\htdocs\`
3. Создайте базу данных `moonlight_db` и импортируйте SQL-файл
4. Запустите Apache и MySQL в XAMPP
5. Откройте в браузере: `http://localhost/moonlight-garden/`
